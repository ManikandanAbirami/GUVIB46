NETLIFY:

1. npm install netlify-cli -g
2. npm run build
3. 'build' folder will be created in the Directory
4. netlify deploy
5. It will ask for the 'site name' and other details. Pls, enter with extra care 
6. It will ask for 'path directory', just type 'build'
7. netlify deploy --prod
